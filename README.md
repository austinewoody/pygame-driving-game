# pygame-driving-game

A simple 2D driving game built with Python and Pygame, showcasing movement, collision logic, and real-time gameplay.

---

## Overview

This project is a simple driving game created with Python and Pygame. It was built to practice core game development concepts such as player movement, obstacle handling, collision detection, and rendering in a real-time game loop.

---

## Features

* Car movement controls
* Real-time game loop
* Obstacle interaction
* Collision detection
* Score or survival-based gameplay
* Simple 2D graphics using Pygame

---

## Tech Stack

* Python
* Pygame

---

## What I Learned

This project helped me practice:

* Game loop design
* Keyboard input handling
* Collision logic
* Object movement
* Basic 2D game development with Pygame

---

## Run the Project

```bash
pip install pygame
python run.py
```
